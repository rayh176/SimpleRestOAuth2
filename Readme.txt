Overview
This task is to create a simple REST API proxy service, that provides a single (unauthenticated) API endpoint that receives JSON data, and calls
an internal (authenticated) API service to perform statistical analysis of that data and return the results. The internal service requires
authentication which is obtained from our OAuth2 identity service.
Technologies
You will be expected to demonstrate a solution using all of the following technologies:
Java
Spring (4.2.4.RELEASE)
OAuth
Rest
